# Power BI Sales & Profit Analysis Dashboard
This project showcases an interactive Power BI dashboard built using datasets from a wholesale wine company based in England. The dashboard provides deep insights into sales performance, profitability, and regional trends, designed for business decision-makers such as grocery stores, restaurants, and pubs. It has been included in this portfolio to highlight expertise in data analysis, visualization, and report design using Power BI.

# Project Overview
This report was developed as part of a Power BI final exam project, demonstrating skills in:

Data modeling and table relationships

DAX calculations and measure creation

Report design and visualization formatting

Interactive data storytelling

# Two main report pages were created:

Sales Dashboard – focuses on total sales, sales trends, and regional performance.

Profits Dashboard – provides insights into profitability by salesperson, supplier, and wine type.

# Key Features
1️⃣ Sales Dashboard

Map Visualization: Displays total cases sold by county (Customer table).

Line Chart: Shows monthly sales trends with smooth interpolation and shadow styling.

Tables: Year and Quarter summary tables with consistent formatting.

Cards: Total Cases Sold, Total Sales and Sales per Salesperson

2️⃣ Profits Dashboard

Data Table: Displays wines with Total Sales, Total Cost, and Net Profit.

Stacked Column Charts: Net Profit by Salesperson and Net Profit by Supplier

Donut Chart: Visualizes Net Profit by Wine Type.
